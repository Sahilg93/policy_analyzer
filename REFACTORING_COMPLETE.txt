KEY FEATURES
================================================================================

✓ Deterministic Calculations
  • No randomness, no probability models, no neural networks
  • Uses only Python math (averages, deltas, multiplication)
  • Fully reproducible results

✓ Zero ML Dependencies
  • No sentence-transformers (embeddings)
  • No sklearn (clustering, similarity)
  • No torch/TensorFlow (neural networks)
  • Pure Python (pandas, numpy, logging)

✓ Bounded Outputs
  • Effects: [-1.0, 1.0]
  • Scores: [-1.0, 1.0]
  • Confidence: [0.0, 1.0]
  • Similarity: [0.0, 1.0]

✓ Graceful Error Handling
  • Missing dates: Skip that analog gracefully
  • No macro data: Return neutral impacts (zeros)
  • Empty corpus: Return empty analog list
  • Never crashes, always returns valid output

✓ Type Hints Throughout
  • Full type annotations in all methods
  • Clear input/output contracts
  • IDE support for autocompletion

✓ Comprehensive Logging
  • INFO level: Major phases and results
  • DEBUG level: Analog processing details
  • WARNING level: Edge cases and recoveries
  • ERROR level: Fatal issues
  • Structured format: [module_name] message

✓ Modular Architecture
  • OutcomeEngine: Isolated impact calculation
  • ScoringEngine: Isolated score calculation
  • AnalogMatcher: Isolated matching
  • Orchestrator: Pure routing, no business logic
  • No dependencies between engines (except data flow)

✓ Unified Output Schema
  • Consistent JSON format for all results
  • Includes: bill metadata, similar_bills, impacts, score, confidence
  • Easy integration with external systems

ARCHITECTURE OVERVIEW
================================================================================

Data Flow:

  Congressional Bills
         ↓
  Ollama AI Classification
         ↓
  build_dataset.py (Orchestrator)
    ├─→ AnalogMatcher (Find similar bills)
    │       ├─ Input: Target policy attributes
    │       └─ Output: Top-K analogs with similarity scores
    │
    ├─→ OutcomeEngine (Estimate macro impacts)
    │       ├─ Input: Analog bills + dates + similarity
    │       ├─ Process: Lookup macro data, calculate changes, aggregate
    │       └─ Output: GDP effect, unemployment effect, matched count
    │
    └─→ ScoringEngine (Calculate net score)
            ├─ Input: Impacts + analytics weights
            ├─ Process: Apply weights, clamp, calculate confidence
            └─ Output: Net score + confidence

                    ↓
              Unified JSON Result
